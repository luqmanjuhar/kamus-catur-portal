Sistem Pendaftaran dan Pengurusan Majlis Sukan Sekolah Daerah

⚠️ Setup Required: Sila setup Google Apps Script untuk simpan data ke Google Sheets.
Semasa ini data hanya disimpan secara tempatan dalam browser.
📋 Panduan Setup Google Apps Script (KLIK SINI)
LANGKAH 1: Buka Google Apps Script
• Pergi ke script.google.com
• Klik "New Project"
• Namakan project: "MSSD Catur Registration"

LANGKAH 2: Salin Kod Apps Script
• Padam semua kod default dalam editor
• Salin dan tampal kod Google Apps Script yang telah disediakan
• Klik "Save" (Ctrl+S)

LANGKAH 3: Update Spreadsheet ID
• Dalam kod, cari baris: const SPREADSHEET_ID = '...'
• Gantikan dengan ID Google Sheets anda
• ID boleh didapati dari URL Google Sheets

LANGKAH 4: Deploy sebagai Web App
• Klik "Deploy" > "New deployment"
• Klik ikon gear ⚙️ dan pilih "Web app"
• Execute as: "Me"
• Who has access: "Anyone"
• Klik "Deploy"

LANGKAH 5: Salin Web App URL
• Salin URL yang diberikan (bermula dengan https://script.google.com/macros/s/...)
• Gantikan dalam kod HTML ini pada baris GOOGLE_APPS_SCRIPT_URL

LANGKAH 6: Test
• Refresh halaman ini
• Cuba daftar satu pendaftaran test
• Semak Google Sheets untuk pastikan data masuk

⚠️ PENTING: Pastikan Google Sheets anda mempunyai 3 sheets dengan nama:
• SEKOLAH - untuk data sekolah
• GURU - untuk data guru
• PELAJAR - untuk data pelajar
